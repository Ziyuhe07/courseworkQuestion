This is Question Setting App User guide

Firstly, load NodeJS server, the name is httpServer.js. Followed by loading phone gap server: type phone gap serve in the terminal. For website, IE browser is recommended. In IE, input address as http://developer.cege.ucl.ac.uk/31271/index.html. And then, the website will be loading. Then, click on the point which is you most interested, and click it. Input the latitude and longitude manually. Think out an interesting question with for possible answers, and make sure one of them are the correct. And then, click "Start Data upload" button. When there is "information inserted" showing on the screen, indicating success. For mobile users, download Phonegap Software and input server address or download by scan QR code. And then follow the guide of web version to fill the table.
